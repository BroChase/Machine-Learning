Machine Learning
Program 4: Fingerprint Classification

***Description**
Implementation of multiclass classifier using svm rbf kernel
Use a training dataset containing 256x256bmp images of fingerprints to train a svm rbf kernel multiclass classifier. 
End result: Accuracy score when tested against unknown fingerprint images 

***To Run***
Py Files:
fingerprint.py
dataproc.py

To test against another set 'setb' change the variable testfingers = 'setb_path'

run fingerprint.py 'main'

Program:
Python 3.6
Written with Pycharm 2017 IDE on a Windows 10 OS PC.
requires:
numpy, pandas, sklearn, scipy
